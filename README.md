# APS UNIP – 3º Semestre

Projeto desenvolvido para a disciplina de **Atividades Práticas Supervisionadas (APS)** do **3º semestre** do curso de **Ciência da Computação** da **Universidade Paulista (UNIP)**. O sistema foi criado com o objetivo de aplicar, de forma prática, os conceitos estudados durante o semestre, utilizando **Java** e os fundamentos de **Programação Orientada a Objetos (POO)**.

## 📚 Sobre o Projeto

Sistema desenvolvido em Java com foco na conscientização ambiental, permitindo ao usuário compreender os impactos do consumo de plástico.

## 🎯 Objetivos do Sistema

- Aplicar conceitos de POO
- Trabalhar lógica de programação
- Promover conscientização ambiental

## 🛠️ Tecnologias Utilizadas

- Java
- NetBeans

## 📁 Estrutura do Projeto

## 💻 Principais Classes

### Main.java
Classe principal responsável por iniciar o sistema e controlar o fluxo do programa.

### CalculadoraConsumo.java
Responsável pelos cálculos relacionados ao consumo de plástico.

### Plastico.java
Onde fica localizado os atributos da Classe Plastico.

## ▶️ Como Executar

1. Clone o repositório
2. Abra no NetBeans
3. Execute o projeto

## 🌱 Conscientização Ambiental

O sistema ajuda a visualizar o impacto do consumo de plástico e incentiva práticas sustentáveis.

## 🎓 Objetivo Acadêmico

Aplicar na prática os conhecimentos do 3º semestre, desenvolvendo um sistema funcional e educativo.
